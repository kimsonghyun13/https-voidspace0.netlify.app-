개인정보 수집 없음 — 완전 안전 앱

개요

VOID는 사용자의 개인정보를 일체 수집하지 않는 완전 오프라인 게임입니다. 이 방침은 앱이 데이터를 어떻게 처리하는지 투명하게 안내합니다.

수집하는 정보

// 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • GPS, IP 기반 위치 정보
  • 기기 고유 식별자 (IMEI, 광고 ID 등)
  • 연락처, 사진, 파일 등 기기 내 데이터
  • 계정 정보 또는 로그인 데이터
  • 결제 또는 금융 정보

// 임시 메모리 데이터 (앱 내부만)

  • 현재 게임 점수 — 세션 종료 시 자동 삭제
  • 최고 점수 — 앱 실행 중에만 유지, 종료 시 초기화
이 데이터는 기기 메모리에만 존재하며, 외부로 전송되거나 저장되지 않습니다.

데이터 사용 방식

  • 사용자 데이터를 제3자와 공유하지 않습니다
  • 광고 목적으로 데이터를 사용하지 않습니다
  • 서버 또는 외부 시스템으로 데이터를 전송하지 않습니다
  • 사용자 행동을 추적하거나 분석하지 않습니다

제3자 서비스

본 앱은 아래 서비스를 포함하지 않습니다:

  • Google Analytics, Firebase 등 분석 도구
  • 광고 네트워크 또는 광고 SDK
  • 소셜 미디어 연동 (Facebook, Kakao 등)
  • 외부 API 호출 또는 네트워크 요청
앱은 완전히 오프라인으로 작동하며 인터넷 연결이 필요하지 않습니다.

아동 개인정보 보호

본 앱은 만 13세 미만 아동을 포함한 모든 연령대가 이용할 수 있는 게임입니다. 아동의 개인정보를 포함하여 어떠한 사용자 데이터도 의도적으로 수집하지 않습니다.

만약 아동으로부터 개인정보가 수집되었다고 판단되면 즉시 삭제 조치합니다.
